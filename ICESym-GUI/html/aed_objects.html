Add, edit and delete objects

Creating Objects
Exists 3 ways to create a new object.
  • Component->New->[object_name]
  • Selecting its icone in the Toolbar
  • Copying for an existing object (select it, press "ctrl+C" and "ctrl+V", or using the Edit menu). This option will copy all the data excepting its connections and it "label".
After, you must complete the form to sets the parameters for Simulation.

Caution!! the defaults will not work, that is, if you can run the simulation results will be unexpected.

Editing Objects
Exists 2 ways to edit a new object.
  • Double click over its image in the diagram.
  • Double click over its label in the objects tree at the left of the screen.


Deleting Objects
Important: Before delete one object you must remove all its connections!
Exists only one way to delete an object: selecting it and press "supr" key.
