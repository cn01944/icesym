Tank



This object model a tank in the simulation.
To access at this form, you can do:
  • Create a new tank with: Components->New->tank
  • Create a new tank choosing in ToolBar the tank's icone
  • Edit an existing tank doing double click over its image object in the simulation diagram
  • Edit an existing tank doing double click over its label in the tree of objects at left of the screen
List of Attributes
  • Label: Choose a label for this object to identifies it in object-tree and in Post Process
  • Number of Nodes: Number of nodes (un-editable, because is equal at the number of tubes connected + the internal node)
  • Deggress of Freedom: Number of degrees of freedom
  • Volume: Volume of tank [m^3]
  • Mass: Mass of gas contained into the tank [kg]
  • H film: Heat-transfer coefficient [W/m^2 K]
  • Area Wall: Tank wall area [m^2]
  • Temperature Wall: Tank wall temperature [K]
  • Cd Ports: Discharge coefficients of the ports -can be imported-
  • State: Initial State. You need to complete it only if Simulation->get_state is set from Here. -can be imported-
  • Connections and own State Box: Switch (0/1) if you want to save the data in each connection for Post Process
  • Save Extras:Switch (0/1) if you want to save extra data for the tank
