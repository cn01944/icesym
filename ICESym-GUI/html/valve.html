Valve



This object model a valve in the simulation.
To access at this form, you can do:
  • Create a new valve with: Components->New->valve
  • Create a new valve choosing in ToolBar the valve's icone
  • Edit an existing valve doing double click over its image object in the simulation diagram
  • Edit an existing valve doing double click over its label in the tree of objects at left of the screen
List of Attributes
  • Label: Choose a label for this object to identifies it in object-tree and in Post Process
  • Type of Valve: Type of valve: intake or exhaust
  • Number of Valves: Number for divide the valve
  • Type Data: Lift valve type:Exponential, Squared sine-wave or User-defined (it actives Lv Define)
  • Open Angle: Valve opening angle [dgs]
  • Close Angle: Valve closing angle [dgs]
  • Diameter Valve: Head valve diameter [m]
  • Lv Max: Maximum valve lift [m]
  • Valve Model:
  • Lv Define: Two-column array with the valve lift data -can be imported-
  • Discharge Coefficient: Discharge coefficient data -can be imported-
