Import Data From Files

ICESym-GUI allows complete the most of arrays with manual input or importing an array from ascii file. The requeried format is the standart octave export format and MUST contains exactly the same number of values that the field to complete.
For example for complete the values of Tube->Temperatures, you can choose constant or linear values, but if you want to use a more specific law, you can import that values. If Tube->nnod = 10, you must choose a ascii file with 10 values qith the next format:

#--------START FILE---------------#
value1
value2
value3
value4
value5
value6
value7
value8
value9
value10
 #--------END FILE---------------#

The previous format is exactly the same as octave export its arrays.
