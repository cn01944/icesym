Junction



This object model a junction in the simulation.
To access at this form, you can do:
  • Create a new junction with: Components->New->junction
  • Create a new junction choosing in ToolBar the junction's icone
  • Edit an existing junction doing double click over its image object in the simulation diagram
  • Edit an existing junction doing double click over its label in the tree of objects at left of the screen
List of Attributes
  • Label: Choose a label for this object to identifies it in object-tree and in Post Process
  • Deggress of Freedom: Number of degrees of freedom
  • Number of Nodes: Number of nodes (un-editable, because is equal at the number of tubes connected)
  • Model Junction: Junction model (for now only one model)
  • Connections Box: Switch (0/1) if you want to save the data in this connectin for Post Process
  • Save Extras:Switch (0/1) if you want to save extra data for the junction
