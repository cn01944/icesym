ICESym-GUI

ICESym-GUI (Internal Combustion Engine Simulator - Graphical User Interface) is a Free Software written in Python and it uses a graphic library called wxPython, that is a blending of the wxWidgets C++ class library with the Python programming language, and is distributed within the package ICESym.

For more information, request, or actualizations, please visit, http://code.google.com/p/icesym/
