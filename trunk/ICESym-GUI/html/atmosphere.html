Atmosphere



This object model a atmosphere in the simulation.
To access at this form, you can do:
  • Create a new atmosphere with: Components->New->atmosphere
  • Create a new atmosphere choosing in ToolBar the atmosphere's icone
  • Edit an existing atmosphere doing double click over its image object in the simulation diagram
List of Attributes:
  • Density: Density in the atmosphere [kg/m^3]
  • Velocity: Velocity in the atmosphere [m/s]
  • Pressure: Pressure in the atmosphere [Pa]
