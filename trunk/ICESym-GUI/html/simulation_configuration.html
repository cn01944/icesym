Simulation Configuration

You can access to Simulation Configuration doing:

Simulation->Configure

In this section you must select the global parameters to the Simulation, for example, you can choose the frequency to save data (and what data you needs) for Post Process -tab I/O Process-, or essencial parameters for the simulation, how the list of rpms to simulate, or the number of strokes per cycle of the engine.
List of Attributes:
  • List of RPMS: List of rpms to simulate separated with ','. Example: 1000,2000
  • Number of cycles for rpm: Number of cycles for each rpm simulated
  • dTheta Rpm: Maximum allowable crank angle variation in each time step [deg]
  • Courant: Courant Number
  • Ga: Specific heat ratio
  • Viscous Flow: Switches on/off (1/0) the friction computation with the pipe wall
  • Heat Flow: Switches on/off (1/0) the heat transfer computation with the pipe wall
  • R Gas: Gas constant
  • Number Stroke: Number of strokes per cycle
  • Engine Type: Engine type:
    • 0 -> Reciprocating
    • 1 -> Opposed-piston
  • Ignition Order: Ignition order of cylinders. Example: 0,1,3,2,6
  • File Save State: File name to save the state in each time step (specified for Iterations per Save)
  • File Save SPD: File name to save the chemical species concentration in each time step (specified for Iterations per Append)
  • Get State From: Select how you initialize the state:
    • From Here-> You must indicates the State in each component
    • From File-> You must indicates the file to load the state for all components. (Used when you need to continue a simulation)
    • From Fortran-> ICESym-1D calculates the initial state numerically with a predictor."
  • find: If you select From File in the previous parameter, you must specified the file that contents the initial state
  • Iterations per Save: Step of iterations for save Simulation State. 0=never. It calculates how number_iteration%ips == 0
  • Iterations per Append: Step of iterations for save the component data. In order to see correctly the results in the post-process you should decrease this value. For higher speed, you should increase this value. 0=never. It calculates how number_iteration%ipa == 0
  • Folder Name: Folder name to save the simulation. In the post-process you must search it. This name is an identifier for this simulation
  • Calculate Engine ... : Calculate engine's data to post-process, such as Power, Torque etc
