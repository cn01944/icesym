Pre-process

In Pre-Process module, you must configure your Simulation.

A lot of attributes is necessary setting. You can do this in easy way, because the Pre-process includes a graphical interfase to modeling your engine. In the same way that the most important engines simulators software's, you can incorporate to the model components doing only one click. Moreover ICESym-GUI allows sets attributes in easiest way, you only must completing the TextBoxes, or selectioning RadioButtons or completing the tables importing data.

In this help section, you can read about what is each attribute of each component to understand how make your own simulation.


Compatibility with ICESym-1D:

You must remember that the simulations saved in ICESym-GUI pre-process automatically can be executed in ICESym-1D.

You only must execute in the ICESym-1d folder:

$ python main.py

and indicates the filename that you saved when the python script was solicited. Preferently you need copy the file to the src path in ICESym-1D.


For more information about run ICESym-1D, see ICESym-1D UserGuide
