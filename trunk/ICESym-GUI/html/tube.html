Tube



This object model a tube in the simulation.
To access at this form, you can do:
  • Create a new tube with: Components->New->tube
  • Create a new tube choosing in ToolBar the tube's icone
  • Edit an existing tube doing double click over its image object in the simulation diagram
  • Edit an existing tube doing double click over its label in the tree of objects at left of the screen
List of Attributes
  • Label: Choose a label for this object to identifies it in object-tree and in Post Process
  • Number of Nodes: Number of nodes
  • Deggress of Freedom: Number of degrees of freedom
  • Length: Pipe length [m]
  • Equispaced: Switch (0/1) if you want to make automatically the array xnod, dividing the length of the pipe in nnod nodes equispaced
  • Has Curvature: Switch (0/1) if you want that the pipe has curvature (0=NO curvature)
  • Xnod: Coordinates in x for each node [m] -can be imported-
  • Diameter: Pipe diameter [m] -can be imported-
  • Twall: Wall pipe temperature [K] -can be imported-
  • Curvature: Curvature in each node along the pipe -can be imported-
  • State: Initial State. You need to complete it only if Simulation->get_state is set from Here. -can be imported-
  • Type Save: you can choose the way to save data (which nodes)
  • Normalized Points: You can choose how many points you want to save. After, you must specify in what normalized positions are that points.
    Example: If the pipe length is 0.2 mts, and you needs save the data in the nodes at 0.0 mts, 0.1 mts and 0.2 mts, you must choose Normalized points = 3 and in the box below, you must input 0 - 0.5 - 1.
